# Power Calculator (累乗計算機)

ブラウザ上で動作する高精度な累乗計算ツールです。  
`BigInt` を活用し、標準の `Number` 型では扱えない巨大な整数の計算を精度を維持したまま行います。

---

## 概要

* **フロントエンド**: React + Vite
* **スタイル**: Tailwind CSS v4
* **計算エンジン**: BigInt（高精度整数） / Number（小数・負の指数）
* **公開**: GitHub Pages（`docs/` 使用）

## 動作フロー

1. **バリデーション**: 入力値の有無、数値形式、指数上限をチェック
2. **安全性の確認**: 結果が約10万桁を超える場合にフリーズ防止のため中断
3. **計算実行**: 
    * 整数同士：`BigInt` で計算
    * 小数・負の指数：標準 `**` 演算にフォールバック
4. **結果出力**: 概算値（指数表記）、正確な桁数、正確な値を生成しUIへ反映

---

## 特徴・機能

* **高精度**: 巨大な整数でも1の位まで正確に算出
* **非同期処理**: `setTimeout` を活用し、重い計算時もUIのフリーズを回避
* **柔軟な表示**: 巨大な値も「指数表記」と「正確な値（1万文字制限）」の両面で出力
* **モダンUI**: Tailwind CSS v4 によるレスポンシブなデザイン

## 仕様・制限事項

* **指数の範囲**: 絶対値 1,000,000 以内
* **計算の上限**: ブラウザのフリーズを防止するため、計算結果が約10万桁を超える場合は実行を制限します。
* **表示の制限**: 正確な値（Exact Result）の出力は先頭 10,000 文字までとし、以降は省略表示となります。
* **対応範囲**: 整数および小数の計算に対応しています（※虚数計算は非対応です）。
